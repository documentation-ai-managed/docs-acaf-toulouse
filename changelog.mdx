## Quick Upgrade Guide

Follow these steps to upgrade Acaf-toulouse to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup before proceeding.

````bash
# Example backup command
pg_dump acaf_toulouse > backup_$(date +%Y%m%d).sql
````

  </Step>
  <Step title="Update Package" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">

````bash
npm install acaf-toulouse@latest
````

      </Tab>
      <Tab title="Docker" icon="docker">

````bash
docker pull acaftoulouse/app:latest
docker-compose up -d
````

      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="database">

    Execute database migrations.

````bash
npm run migrate
````

  </Step>
  <Step title="Verify Installation" icon="check-circle">

    Check the dashboard at `https://dashboard.example.com` and test key features.

  </Step>
</Steps>
